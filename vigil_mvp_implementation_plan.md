## 1. Project Overview

**Vigil** is a real-time global attention inequality platform. It computes a single score called the **Invisible Index** for every country on Earth every 15 minutes. The index measures how much humanitarian suffering is happening in a country versus how much media attention that country is receiving. A high index means the world is ignoring that crisis.

**Core formula:**
```
Invisible Index = Suffering Score / Attention Score
```

- **Suffering Score** — composite of ACLED conflict data, UNHCR displacement numbers, WHO disease outbreaks
- **Attention Score** — count of crisis-specific news articles from GDELT, filtered through a Claude classification pass

The result is displayed on an interactive 3D globe. Every country glows based on its Invisible Index. A user clicks any country and receives a plain-English crisis brief generated by Claude and read aloud by ElevenLabs.

## 2. MVP Scope

### In Scope
- DataAgent: pulls external APIs, runs Claude classification, computes Invisible Index, writes to MongoDB
- FactCheckerAgent: validates each DataAgent cycle before it propagates
- AlertAgent: detects score spikes, fires webhook to FastAPI
- QueryAgent: implements Fetch.ai ChatProtocol, answers natural language queries via ASI:One (OmegaClaw prize track)
- FastAPI backend: serves globe data, generates crisis briefs on demand, receives alert webhooks
- React frontend: 3D globe with Three.js, country click → crisis brief, ElevenLabs voice playback
- MongoDB Atlas: stores country profiles, index snapshots, quarantine log, alert log

### Out of Scope (post-MVP)
- Roblox Signal Run game
- World U attention pledge system
- MCP server for AI coding agents
- Cloudinary media management
- Gemma on-device offline mode
- Email/push notifications to external journalists
- RSF Press Freedom, FAO AQUASTAT, IMF layers (only wars, displacement, famine, disease in MVP)

## 5. Data Models

### MongoDB Collections

#### `countries` collection
One document per country. Updated every 15 minutes by DataAgent (via FactChecker).

```json
{
  "_id": "AGO",
  "name": "Angola",
  "iso3": "AGO",
  "lat": -11.2,
  "lng": 17.87,
  "invisible_index": 15847.3,
  "suffering_score": 7230.1,
  "attention_score": 0.456,
  "article_count_raw": 12,
  "article_count_filtered": 2,
  "conflict_events": 14,
  "displaced_persons": 7300000,
  "disease_outbreaks": 1,
  "last_updated": "2026-04-25T03:15:00Z",
  "data_sources": ["ACLED", "UNHCR", "WHO", "GDELT"]
}
```

#### `index_snapshots` collection
Time-series. One document per country per cycle.

```json
{
  "country_code": "AGO",
  "invisible_index": 15847.3,
  "suffering_score": 7230.1,
  "attention_score": 0.456,
  "timestamp": "2026-04-25T03:15:00Z",
  "cycle_id": "20260425-0315"
}
```

#### `quarantine_log` collection
Written by FactCheckerAgent when it rejects data.

```json
{
  "cycle_id": "20260425-0315",
  "country_code": "TCD",
  "rejected_score": 999999.0,
  "last_known_good_score": 12340.5,
  "reasons": ["score exceeds historical ceiling of 80000"],
  "claude_review": false,
  "timestamp": "2026-04-25T03:15:04Z"
}
```

#### `alert_log` collection
Written by FastAPI when AlertAgent fires a webhook.

```json
{
  "country_code": "SDN",
  "previous_score": 8000.0,
  "new_score": 21500.0,
  "delta": 13500.0,
  "fired_at": "2026-04-25T03:15:10Z"
}
```

### uAgent Message Models (`agents/shared/models.py`)

```python
from uagents import Model
from typing import Dict, List, Optional

class RawDataPayload(Model):
    """DataAgent → FactCheckerAgent"""
    countries: Dict[str, float]           # country_code → invisible_index
    article_counts_raw: Dict[str, int]    # country_code → raw GDELT article count
    article_counts_filtered: Dict[str, int] # country_code → post-Claude count
    suffering_scores: Dict[str, float]    # country_code → suffering component
    attention_scores: Dict[str, float]    # country_code → attention component
    cycle_id: str                         # e.g. "20260425-0315"

class VerifiedIndexUpdate(Model):
    """FactCheckerAgent → AlertAgent"""
    countries: Dict[str, float]           # verified country_code → invisible_index
    quarantined: List[str]                # country codes that were rejected this cycle
    cycle_id: str

class AlertPayload(Model):
    """AlertAgent → FastAPI webhook (HTTP POST body)"""
    country_code: str
    country_name: str
    previous_score: float
    new_score: float
    delta: float
    cycle_id: str
```

## 6. Agent 1: DataAgent

**File:** `agents/data_agent.py`  
**Deployment:** Agentverse hosted agent  
**Trigger:** `@agent.on_interval(period=900.0)` — every 15 minutes  
**Responsibility:** Pull all external APIs, run Claude classification on GDELT results, compute Invisible Index for all countries, write verified data to MongoDB, send raw payload to FactCheckerAgent.

### 6.1 GDELT Pull

```python
def fetch_gdelt_articles() -> List[dict]:
    """
    Fetch recent news articles from GDELT v2 Doc API.
    Returns list of article dicts with keys: title, url, country_code, sourcecountry.
    
    GDELT endpoint: GET https://api.gdeltproject.org/api/v2/doc/doc
    Required params:
        query       = "humanitarian crisis OR famine OR conflict OR displacement OR outbreak"
        mode        = "ArtList"
        maxrecords  = 250
        format      = "json"
        timespan    = "15min"    ← only articles from the last 15 minutes
        sort        = "DateDesc"
    
    Returns: list of article objects from response["articles"]
    If response["articles"] is empty or key is missing, return []
    Log a warning if total count is 0 (indicates possible API failure).
    Do NOT raise — return [] and let FactChecker handle the empty case.
    """
```

### 6.2 Claude Classification Pass

```python
def classify_articles(articles: List[dict]) -> Dict[str, int]:
    """
    For each article, call Claude to determine:
    1. Is this article primarily about a humanitarian crisis?
    2. Which country is the crisis in?
    
    Returns: Dict[country_code, filtered_article_count]
    
    Implementation rules:
    - Use model: claude-haiku-4-5-20251001  (NOT Sonnet — cost control)
    - Batch articles in groups of 10 per API call to reduce HTTP overhead
    - Prompt per batch:
        "For each article title below, answer with a JSON array.
         Each element: {"idx": <index>, "crisis": true/false, "country_iso3": "<ISO3 or null>"}
         Only mark crisis=true if the article is primarily about humanitarian suffering
         (conflict, famine, displacement, disease outbreak, natural disaster).
         Sports, politics, economics without humanitarian angle = false.
         Titles: <numbered list>"
    - Parse the JSON array from Claude response
    - If Claude response is not valid JSON, skip that batch (do not crash)
    - Accumulate country_code → count from all batches
    - Cap total Claude calls at 25 batches (250 articles max per cycle)
    
    Returns empty dict {} on complete failure — FactChecker will catch this.
    """
```

### 6.3 Suffering Score APIs

```python
def fetch_acled_data() -> Dict[str, dict]:
    """
    Fetch conflict event data from ACLED API.
    
    Endpoint: GET https://api.acleddata.com/acled/read
    Params:
        key         = ACLED_API_KEY
        email       = ACLED_EMAIL
        limit       = 5000
        fields      = "country,iso3,fatalities,event_date"
        event_date  = "<today - 30 days>|<today>"  (pipe-separated range)
        format      = "json"
    
    Aggregate by iso3: sum fatalities, count events.
    Returns: Dict[iso3, {"fatalities": int, "events": int}]
    On HTTP error or timeout (>10s): return {} and log warning.
    """

def fetch_unhcr_data() -> Dict[str, int]:
    """
    Fetch displacement (refugee + IDP) counts from UNHCR.
    
    Endpoint: GET https://data.unhcr.org/population/get/timeseries
    Params:
        coo_all     = true
        limit       = 300
        year        = <current year>
        type        = "persons"
    
    Returns: Dict[iso3, total_displaced_persons]
    On error: return {}
    """

def fetch_who_outbreaks() -> Dict[str, int]:
    """
    Fetch active disease outbreak counts from WHO.
    
    Endpoint: GET https://www.who.int/api/news/outbreak-news
    Params: None required for public endpoint.
    
    Parse response for country mentions.
    Returns: Dict[iso3, outbreak_count]
    Note: WHO API structure may require HTML scraping as fallback.
    If WHO API is unavailable, return {} — do not crash the cycle.
    """
```

### 6.4 Invisible Index Computation

```python
def compute_invisible_index(
    acled: Dict[str, dict],
    unhcr: Dict[str, int],
    who: Dict[str, int],
    gdelt_filtered: Dict[str, int],
    gdelt_raw: Dict[str, int],
    all_countries: List[str]  # ISO3 codes for all 195 countries
) -> Dict[str, dict]:
    """
    For each country in all_countries:
    
    suffering_score = (
        (fatalities * 10) +
        (conflict_events * 5) +
        (displaced_persons / 1000) +
        (outbreak_count * 500)
    )
    
    attention_score = max(filtered_article_count, 0.001)
    # Floor attention at 0.001 to prevent division by zero
    # 0.001 represents "near-zero coverage" not "exactly zero"
    
    invisible_index = suffering_score / attention_score
    
    If suffering_score == 0: invisible_index = 0
    (No suffering = no crisis to cover, index is 0 regardless of attention)
    
    Returns: Dict[iso3, {
        "invisible_index": float,
        "suffering_score": float,
        "attention_score": float,
        "article_count_raw": int,
        "article_count_filtered": int,
        "conflict_events": int,
        "displaced_persons": int,
        "disease_outbreaks": int
    }]
    """
```

### 6.5 MongoDB Write (Atlas Data API via HTTP)

```python
def write_to_mongo(computed: Dict[str, dict], cycle_id: str):
    """
    Write computed index values to MongoDB Atlas using the Data API (HTTP).
    Do NOT use pymongo — it is not available in Agentverse hosted environments.
    
    For each country:
    1. Upsert into `countries` collection:
       Filter: {"_id": iso3}
       Update: {"$set": {all fields + last_updated + data_sources}}
       upsert: true
    
    2. Insert into `index_snapshots` collection:
       Document: {country_code, invisible_index, suffering_score, attention_score, timestamp, cycle_id}
    
    HTTP endpoint for upsert:
    POST {MONGO_DATA_API_URL}/action/updateOne
    Headers: {"api-key": MONGO_DATA_API_KEY, "Content-Type": "application/json"}
    
    HTTP endpoint for insert:
    POST {MONGO_DATA_API_URL}/action/insertOne
    
    Batch snapshot inserts using /action/insertMany — one call for all 195 countries.
    Do NOT loop 195 individual insertOne calls — too slow.
    
    On HTTP error: log the error, do not crash. The cycle continues.
    """
```

### 6.6 Full DataAgent Loop

```python
from uagents import Agent, Context
import os, json, datetime, requests

FACT_CHECKER_ADDRESS = os.environ["FACT_CHECKER_AGENT_ADDRESS"]

agent = Agent(name="vigil_data_agent", seed=os.environ["DATA_AGENT_SEED"])

ALL_COUNTRIES = [
    # Full list of 195 ISO3 country codes
    "AFG", "AGO", "ALB", ... "ZWE"
]

@agent.on_interval(period=900.0)
async def update_globe(ctx: Context):
    cycle_id = datetime.datetime.utcnow().strftime("%Y%m%d-%H%M")
    ctx.logger.info(f"Starting cycle {cycle_id}")
    
    # 1. Fetch all data sources in parallel where possible
    articles      = fetch_gdelt_articles()
    acled         = fetch_acled_data()
    unhcr         = fetch_unhcr_data()
    who           = fetch_who_outbreaks()
    
    # 2. Claude classification pass
    gdelt_filtered = classify_articles(articles)
    gdelt_raw      = {a.get("country_code", "UNK"): 1 for a in articles}  # approximate raw count
    
    # 3. Compute index for all countries
    computed = compute_invisible_index(acled, unhcr, who, gdelt_filtered, gdelt_raw, ALL_COUNTRIES)
    
    # 4. Write to MongoDB
    write_to_mongo(computed, cycle_id)
    
    # 5. Send raw payload to FactCheckerAgent for validation
    payload = RawDataPayload(
        countries={k: v["invisible_index"] for k, v in computed.items()},
        article_counts_raw={k: v["article_count_raw"] for k, v in computed.items()},
        article_counts_filtered={k: v["article_count_filtered"] for k, v in computed.items()},
        suffering_scores={k: v["suffering_score"] for k, v in computed.items()},
        attention_scores={k: v["attention_score"] for k, v in computed.items()},
        cycle_id=cycle_id
    )
    await ctx.send(FACT_CHECKER_ADDRESS, payload)
    ctx.logger.info(f"Cycle {cycle_id} complete. Sent to FactChecker.")

if __name__ == "__main__":
    agent.run()
```

## 7. Agent 2: FactCheckerAgent

**File:** `agents/fact_checker_agent.py`  
**Deployment:** Agentverse hosted agent  
**Trigger:** `@agent.on_message(model=RawDataPayload)` — fires when DataAgent sends data  
**Responsibility:** Validate each DataAgent cycle. Reject anomalous scores. Quarantine bad data. Pass only verified data to AlertAgent.

### 7.1 Validation Rules

```python
HISTORICAL_MAX_INDEX = 80000.0
# No country has ever realistically exceeded this. Angola's worst is ~16K.
# Anything above 80K is almost certainly a computation artifact.

MAX_CYCLE_MULTIPLIER = 5.0
# A legitimate crisis does not escalate 5x in 15 minutes.
# If score changes more than 5x up or down vs last cycle, flag it.

MIN_GLOBAL_ARTICLES = 5
# If GDELT returned fewer than 5 filtered articles globally, the API call likely failed.
# Reject the entire cycle in this case — do not overwrite last known good data.

MAX_CLAUDE_REVIEW_COUNTRIES = 5
# Only call Claude for plausibility review if <= 5 countries are flagged.
# More than 5 flagged countries means a systemic data problem — skip Claude, reject batch.
```

### 7.2 Full FactCheckerAgent

```python
from uagents import Agent, Context
import os, json, datetime, requests

ALERT_AGENT_ADDRESS = os.environ["ALERT_AGENT_ADDRESS"]
MONGO_DATA_API_URL = os.environ["MONGO_DATA_API_URL"]
MONGO_DATA_API_KEY = os.environ["MONGO_DATA_API_KEY"]
ANTHROPIC_API_KEY = os.environ["ANTHROPIC_API_KEY"]

agent = Agent(name="vigil_fact_checker", seed=os.environ["FACT_CHECKER_SEED"])

@agent.on_message(model=RawDataPayload)
async def fact_check(ctx: Context, sender: str, msg: RawDataPayload):
    verified = {}
    quarantined = []
    
    # Load last verified scores from agent storage
    raw_last = ctx.storage.get("last_verified_scores")
    last_scores = json.loads(raw_last) if raw_last else {}
    
    # Global check: did GDELT return basically nothing?
    total_filtered = sum(msg.article_counts_filtered.values())
    if total_filtered < MIN_GLOBAL_ARTICLES:
        ctx.logger.warning(
            f"Cycle {msg.cycle_id} rejected: only {total_filtered} filtered articles globally. "
            "Likely GDELT API failure. Not propagating."
        )
        return  # AlertAgent never sees this cycle
    
    # Per-country checks
    for country, score in msg.countries.items():
        reasons = []
        
        # Check 1: Hard ceiling
        if score > HISTORICAL_MAX_INDEX:
            reasons.append(f"score {score:.1f} exceeds ceiling {HISTORICAL_MAX_INDEX}")
        
        # Check 2: Cycle-over-cycle multiplier
        if country in last_scores and last_scores[country] > 0:
            ratio = score / last_scores[country]
            if ratio > MAX_CYCLE_MULTIPLIER or ratio < (1.0 / MAX_CYCLE_MULTIPLIER):
                reasons.append(
                    f"score changed {ratio:.2f}x vs last cycle "
                    f"({last_scores[country]:.1f} → {score:.1f})"
                )
        
        if reasons:
            quarantined.append({
                "country": country,
                "score": score,
                "last_known_good": last_scores.get(country),
                "reasons": reasons
            })
            # Use last known good score if available
            if country in last_scores:
                verified[country] = last_scores[country]
            # If no history exists, do not include this country in output at all
        else:
            verified[country] = score
    
    # Claude plausibility review for small flagged sets
    if 0 < len(quarantined) <= MAX_CLAUDE_REVIEW_COUNTRIES:
        flagged_str = json.dumps([
            {"country": q["country"], "score": q["score"], "reasons": q["reasons"]}
            for q in quarantined
        ], indent=2)
        
        try:
            resp = requests.post(
                "https://api.anthropic.com/v1/messages",
                headers={
                    "x-api-key": ANTHROPIC_API_KEY,
                    "anthropic-version": "2023-06-01",
                    "content-type": "application/json"
                },
                json={
                    "model": "claude-haiku-4-5-20251001",
                    "max_tokens": 150,
                    "messages": [{
                        "role": "user",
                        "content": (
                            "The following countries were flagged as anomalous in a humanitarian "
                            "crisis index update. Based on current world events you know about, "
                            "which of these scores might actually be plausible right now? "
                            "Reply with ONLY a JSON array of ISO3 country codes that are plausible. "
                            "Example: [\"AGO\", \"SDN\"]. If none are plausible, reply [].\n\n"
                            f"{flagged_str}"
                        )
                    }]
                },
                timeout=10
            ).json()
            
            plausible_text = resp["content"][0]["text"].strip()
            plausible_codes = json.loads(plausible_text)
            
            # Restore plausible flagged countries to verified
            for item in quarantined[:]:
                if item["country"] in plausible_codes:
                    verified[item["country"]] = item["score"]
                    quarantined.remove(item)
                    ctx.logger.info(
                        f"Claude restored {item['country']} score {item['score']:.1f} as plausible"
                    )
        except Exception as e:
            ctx.logger.warning(f"Claude plausibility review failed: {e}. Keeping quarantine decisions.")
    
    # Write quarantine log to MongoDB
    if quarantined:
        try:
            requests.post(
                f"{MONGO_DATA_API_URL}/action/insertMany",
                headers={"api-key": MONGO_DATA_API_KEY, "Content-Type": "application/json"},
                json={
                    "collection": "quarantine_log",
                    "database": os.environ["MONGO_DATABASE"],
                    "dataSource": os.environ["MONGO_DATASOURCE"],
                    "documents": [
                        {
                            "cycle_id": msg.cycle_id,
                            "country_code": q["country"],
                            "rejected_score": q["score"],
                            "last_known_good_score": q.get("last_known_good"),
                            "reasons": q["reasons"],
                            "timestamp": datetime.datetime.utcnow().isoformat() + "Z"
                        }
                        for q in quarantined
                    ]
                },
                timeout=10
            )
        except Exception as e:
            ctx.logger.warning(f"Failed to write quarantine log: {e}")
    
    # Save verified scores as the new baseline for next cycle
    ctx.storage.set("last_verified_scores", json.dumps(verified))
    
    ctx.logger.info(
        f"Cycle {msg.cycle_id}: {len(verified)} verified, {len(quarantined)} quarantined"
    )
    
    # Forward to AlertAgent
    await ctx.send(ALERT_AGENT_ADDRESS, VerifiedIndexUpdate(
        countries=verified,
        quarantined=[q["country"] for q in quarantined],
        cycle_id=msg.cycle_id
    ))

if __name__ == "__main__":
    agent.run()
```

## 8. Agent 3: AlertAgent

**File:** `agents/alert_agent.py`  
**Deployment:** Agentverse hosted agent  
**Trigger:** `@agent.on_message(model=VerifiedIndexUpdate)` — fires when FactChecker sends verified data  
**Responsibility:** Compare verified scores to previous cycle. Detect score spikes. Fire webhook to FastAPI when a spike is detected. Store last scores in agent storage.

### 8.1 Spike Detection Rules

```python
SPIKE_THRESHOLD_ABSOLUTE = 3000.0
# A country's index must rise by at least 3000 points to trigger an alert.
# Prevents noise from small fluctuations.

SPIKE_THRESHOLD_RELATIVE = 2.0
# A country's index must also at least double vs last cycle to trigger.
# Both conditions must be met.

TOP_N_TO_SURFACE = 3
# The AlertAgent always surfaces the top 3 most-ignored countries each cycle
# regardless of spike — this powers the QueryAgent response and the frontend banner.
```

### 8.2 Full AlertAgent

```python
from uagents import Agent, Context
import os, json, datetime, requests

BACKEND_URL = os.environ["BACKEND_URL"]
INTERNAL_WEBHOOK_SECRET = os.environ["INTERNAL_WEBHOOK_SECRET"]

agent = Agent(name="vigil_alert_agent", seed=os.environ["ALERT_AGENT_SEED"])

@agent.on_message(model=VerifiedIndexUpdate)
async def handle_verified_update(ctx: Context, sender: str, msg: VerifiedIndexUpdate):
    # Load previous scores
    raw_last = ctx.storage.get("last_scores")
    last_scores = json.loads(raw_last) if raw_last else {}
    
    spikes = []
    for country, new_score in msg.countries.items():
        old_score = last_scores.get(country, 0)
        delta = new_score - old_score
        relative_change = (new_score / old_score) if old_score > 0 else float("inf")
        
        if delta >= SPIKE_THRESHOLD_ABSOLUTE and relative_change >= SPIKE_THRESHOLD_RELATIVE:
            spikes.append({
                "country_code": country,
                "previous_score": old_score,
                "new_score": new_score,
                "delta": delta,
                "cycle_id": msg.cycle_id
            })
    
    # Compute top N most ignored (for QueryAgent and frontend banner)
    sorted_countries = sorted(msg.countries.items(), key=lambda x: x[1], reverse=True)
    top_ignored = [{"country_code": c, "score": s} for c, s in sorted_countries[:TOP_N_TO_SURFACE]]
    ctx.storage.set("top_ignored", json.dumps(top_ignored))
    
    # Save current scores as baseline
    ctx.storage.set("last_scores", json.dumps(msg.countries))
    
    # Fire webhook to FastAPI for each spike
    for spike in spikes:
        try:
            requests.post(
                f"{BACKEND_URL}/internal/alert",
                headers={
                    "Content-Type": "application/json",
                    "X-Webhook-Secret": INTERNAL_WEBHOOK_SECRET
                },
                json=spike,
                timeout=8
            )
            ctx.logger.info(f"Alert fired for {spike['country_code']}: +{spike['delta']:.0f}")
        except Exception as e:
            ctx.logger.warning(f"Failed to fire alert webhook for {spike['country_code']}: {e}")
    
    if not spikes:
        ctx.logger.info(f"Cycle {msg.cycle_id}: no spikes detected. Top ignored: {top_ignored[0]['country_code']} ({top_ignored[0]['score']:.0f})")

if __name__ == "__main__":
    agent.run()
```

### 9.2 Full QueryAgent

```python
from uagents import Agent, Context, Protocol
from uagents_core.contrib.protocols.chat import (
    ChatAcknowledgement,
    ChatMessage,
    EndSessionContent,
    TextContent,
    chat_protocol_spec,
)
import os, json, requests
from datetime import datetime
from uuid import uuid4

ANTHROPIC_API_KEY = os.environ["ANTHROPIC_API_KEY"]
MONGO_DATA_API_URL = os.environ["MONGO_DATA_API_URL"]
MONGO_DATA_API_KEY = os.environ["MONGO_DATA_API_KEY"]

agent = Agent(name="vigil_query_agent", seed=os.environ["QUERY_AGENT_SEED"])
chat_proto = Protocol(spec=chat_protocol_spec)

def get_top_ignored(n: int = 5) -> list:
    """Fetch top N most ignored countries from MongoDB."""
    try:
        resp = requests.post(
            f"{MONGO_DATA_API_URL}/action/find",
            headers={"api-key": MONGO_DATA_API_KEY, "Content-Type": "application/json"},
            json={
                "collection": "countries",
                "database": os.environ["MONGO_DATABASE"],
                "dataSource": os.environ["MONGO_DATASOURCE"],
                "filter": {"invisible_index": {"$gt": 0}},
                "sort": {"invisible_index": -1},
                "limit": n,
                "projection": {"_id": 1, "name": 1, "invisible_index": 1,
                               "displaced_persons": 1, "conflict_events": 1,
                               "article_count_filtered": 1, "last_updated": 1}
            },
            timeout=8
        ).json()
        return resp.get("documents", [])
    except Exception:
        return []

def get_country_data(iso3: str) -> dict:
    """Fetch a single country's current data from MongoDB."""
    try:
        resp = requests.post(
            f"{MONGO_DATA_API_URL}/action/findOne",
            headers={"api-key": MONGO_DATA_API_KEY, "Content-Type": "application/json"},
            json={
                "collection": "countries",
                "database": os.environ["MONGO_DATABASE"],
                "dataSource": os.environ["MONGO_DATASOURCE"],
                "filter": {"_id": iso3.upper()}
            },
            timeout=8
        ).json()
        return resp.get("document", {})
    except Exception:
        return {}

def generate_response(user_query: str) -> str:
    """Call Claude to generate a natural language response from live Vigil data."""
    top5 = get_top_ignored(5)
    data_context = json.dumps(top5, indent=2, default=str)
    
    try:
        resp = requests.post(
            "https://api.anthropic.com/v1/messages",
            headers={
                "x-api-key": ANTHROPIC_API_KEY,
                "anthropic-version": "2023-06-01",
                "content-type": "application/json"
            },
            json={
                "model": "claude-haiku-4-5-20251001",
                "max_tokens": 300,
                "system": (
                    "You are Vigil, a humanitarian crisis intelligence assistant. "
                    "You have access to real-time data showing which global crises the world is ignoring. "
                    "The Invisible Index = Suffering Score / Attention Score. Higher = more ignored. "
                    "Answer in plain English. Be factual. No jargon. 2-3 sentences maximum. "
                    "Always cite the Invisible Index score when naming a country."
                ),
                "messages": [
                    {"role": "user", "content": (
                        f"Live data (top most-ignored crises right now):\n{data_context}\n\n"
                        f"User query: {user_query}"
                    )}
                ]
            },
            timeout=15
        ).json()
        return resp["content"][0]["text"]
    except Exception as e:
        return f"Unable to retrieve live data at this time. Error: {str(e)}"

@chat_proto.on_message(ChatMessage)
async def handle_chat(ctx: Context, sender: str, msg: ChatMessage):
    # Acknowledge receipt
    await ctx.send(sender, ChatAcknowledgement(
        timestamp=datetime.utcnow(),
        acknowledged_msg_id=msg.msg_id
    ))
    
    # Extract text from message
    user_query = ""
    for item in msg.content:
        if hasattr(item, "text"):
            user_query = item.text
            break
    
    if not user_query:
        response_text = "Please ask a question about the current global humanitarian situation."
    else:
        response_text = generate_response(user_query)
    
    # Send response
    await ctx.send(sender, ChatMessage(
        msg_id=str(uuid4()),
        timestamp=datetime.utcnow(),
        content=[TextContent(type="text", text=response_text)]
    ))
    
    # End session
    await ctx.send(sender, ChatMessage(
        msg_id=str(uuid4()),
        timestamp=datetime.utcnow(),
        content=[EndSessionContent(type="end-session")]
    ))

agent.include(chat_proto, publish_manifest=True)

if __name__ == "__main__":
    agent.run()
```

## 16. MVP Cut Decisions

These decisions were made deliberately. Do not add them back without finishing the core pipeline first.

| Feature | Decision | Reason |
|---|---|---|
| Email notifications to journalists | Cut | Agentverse has no push-to-external system. Requires third-party email service. Out of scope for 36h. |
| pymongo in agents | Cut | Not available in Agentverse hosted environments. All DB calls use Atlas Data API over HTTP. |
| ActionAgent as Agentverse agent | Cut | Latency-sensitive user-facing call. FastAPI → Claude directly is faster and simpler. |
| Anthropic Python SDK in agents | Cut | Not available in Agentverse hosted environments. All Claude calls use raw `requests`. |
| All 9 globe data layers | Cut | MVP has 4: conflict (ACLED), displacement (UNHCR), disease (WHO), and master composite. |
| Roblox Signal Run | Cut | Separate 20+ hour build. Qualifies for separate prize. Build after MVP is stable. |
| World U attention pledges | Cut | Requires World U API integration and token system. Post-MVP. |
| ElevenLabs streaming audio | Cut | Return base64 audio in response body. Streaming adds complexity for minimal UX gain. |
| Cloudinary media management | Cut | Frontend loads country flag images from a free CDN or public domain source. |
| Historical trend chart in frontend | Cut | Pre-populated mock data in MongoDB for demo countries. Real history accumulates over time. |
